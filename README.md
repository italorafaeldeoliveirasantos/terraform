# Terraform build to a AWS infra and a nginx based app

## To get things going
Set this variables
```sh 
export AWS_ACCESS_KEY_ID="anaccesskey"
export AWS_SECRET_ACCESS_KEY="asecretkey"
export AWS_DEFAULT_REGION="us-west-2"
```

## Initialize your working dir
```sh 
terraform init
```

## Apply the plan
```sh
terraform apply
```
